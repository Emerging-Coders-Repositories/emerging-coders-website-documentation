# Contribution Guide for Emerging Coders Website

Thank you for your interest in contributing to the Emerging Coders Website! Your contributions are crucial in making the project a success. This guide provides the necessary steps to contribute to the project seamlessly. Please follow the steps below to ensure a smooth contribution process.

## Prerequisites
- A GitHub account.
- Basic understanding of Git and GitHub.
- [Git installed](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) on your local machine.

## Contribution Workflow

### Step 1: Fork the Repository
1. Navigate to the [Emerging Coders Website Repository](https://github.com/ethanpaneraa/Emerging-Coders-Website).
2. Click on the **Fork** button at the top-right corner of the page.
3. Select your GitHub profile to create a fork of the repository.

### Step 2: Clone the Forked Repository
1. Go to your forked repository on GitHub.
2. Click on the **Code** button, then copy the HTTPS URL.
3. Open a terminal on your local machine, navigate to the directory where you want to clone the repository.
4. Run the following command:

```bash
git clone [URL]
```

Replace `[URL]` with the HTTPS URL you copied earlier.

### Step 3: Create a New Branch
1. Navigate into the cloned repository directory:

```bash
cd Emerging-Coders-Website
```

2. Create a new branch for the feature or bug you are working on:

```bash
git checkout -b [branch-name]
```

Replace `[branch-name]` with a descriptive name for your branch (e.g., `feature/new-feature-name` or `bug/bug-description`).

### Step 4: Make Your Changes
1. Make the necessary changes to the app using your favorite code editor.
2. Test your changes thoroughly to ensure they work as expected and do not introduce new bugs.

### Step 5: Commit and Push Your Changes
1. Stage your changes:

```bash
git add .
```

2. Commit your changes:

```bash
git commit -m "Describe your changes here"
```

3. Push your changes to your forked repository:

```bash
git push origin [branch-name]
```

### Step 6: Create a Pull Request
1. Go to the [Emerging Coders Website Repository](https://github.com/ethanpaneraa/Emerging-Coders-Website).
2. Click on **Pull requests** tab.
3. Click on the **New Pull Request** button.
4. Select your fork and the branch you worked on.
5. Fill in the PR title and description. Describe the changes you made, why you made them, any issues that are related or resolved by your PR, and any additional information that will help maintainers understand and review your contribution.
6. Click **Create Pull Request**.

## Conclusion
Thank you for following this contribution guide. Your effort in contributing to the Emerging Coders Website is highly appreciated. Together, we can build a project that is beneficial to the community and beyond.
