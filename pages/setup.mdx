# Setting Up the Website Locally


Here's a guide on how to set up and install all of the necessary dependencies like Node.js and npm to run the website locally on your computer. This guide will also describe how to run the website locally on your computer. There are sections for Windows, Mac, and Linux. The instructions for installing everything are compiled based on the latest methods and documentation for each operating system. If you have any issues, please let me know and I will try to help you out.
